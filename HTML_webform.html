HTML 문서 구조화

기존 HTML의 한계 : 문서 구조를 표현하는 태그가 없어 소스를 통해 문서 구조 파악이 힘듬
->시맨틱 태그를 통해 웹 문서를 구조화할 수 있다!
<header>, <section>, <article>, <main>, <summary>, <mark>, <time>...등등

구조화된 태그들의 위치와 색, 모양등은 개발자가 CSS를 사용해 지정한다.

시맨틱 블록 태그

시맨틱 태그를 사용해 블록을 구성할 수 있다.

<FIGURE>

<figure>태그는 본문에 삽입된 이미지, 동영상, 소스코드 등 콘텐츠를 블록화한다.
블록의 제목은 <figure>태그 내의 <figcaption>태그로 작성한다.

<DETAILS>와 <SUMMARY>활용

<details>로 만든 블록에 핸들(▶)이 나타난다.

<summary>태그는 <details>로 구성되는 블록의 제목을 표현한다.

웹 폼

    <form>태그 속성
  • name : 폼의 이름을 지정하는 속성으로 자바스크립트에서 사용
  • action : 폼 데이터를 처리할 웹 서버 응용프로그램 지정. submit버튼이 클릭되면 action에 지정된 웹 서버 응용프로그램을 실행한다.
  • method : 폼 데이터를 웹 서버로 전송하는 방식. 대표적으로 GET, POST가 있다.

    버튼 기능의 종류
  • button : 아무 기능이 없는 버튼. 자바스크립트에서 사용

  • reset : 폼에 입력된 내용을 초기화
