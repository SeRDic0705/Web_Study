CSS3 스타일 시트 구성

  • 셀렉터 : CSS스타일시트를 HTML페이지에 적용하도록 만든 이름
  • 프로퍼티 : 스타일 속성 이름. CSS3에는 약 200개 정도의 프로퍼티가 있다.
  • 값 : 프로퍼티의 값
  • 주석문 : /*...*/주석문. 여러 줄에 걸쳐 아무 위치나 사용 가능
*셀렉터, 프로퍼티, 값은 대소문자를 구분하지 않는다.

CSS3 스타일 시트 만들기

    HTML문서에 CSS스타일시트 만드는 3가지 방법
  • <style>태그에 스타일시트 작성 <style>태그는 <head>태그 내에서만 작성 가능하며, 작성된 스타일시트는 웹 페이지 전체에 적용된다.
  • style 속성에 스타일시트 작성
  • 스타일시트를 외부 파일로 작성 후, 파일을 불러와 사용 두가지 방법 사용 가능
    <link href="filename.css" type="text/css" rel=stylesheet">
    @import url("filename.css")

CSS3 규칙

-스타일 상속

부모태그
자신을 둘러싸는 태그. 부모 태그로부터 스타일시트를 상속받는다

<p>태그의 스타일시트 color:green
<em>은 <p>의 자식이다.


-스타일 합치기와 오버라이딩

    각 태그에는 4가지 종류의 스타일시트가 동시에 적용될 수 있다
  1. 브라우저의 디폴트 스타일시트
  2. .css 스타일시트 파일에 작성된 스타일시트
  3. <style>태그에 작성된 스타일시트
  4. style속성에 작성된 스타일시트
*아래로 내려갈수록 우선순위가 높다.

<style>에서 작성한 ivory색 배경보다 <p>의 style속성에 작성한 orange색 배경이 우선 적용된다.
