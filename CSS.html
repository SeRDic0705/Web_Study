CSS3 스타일 시트 구성

  • 셀렉터 : CSS스타일시트를 HTML페이지에 적용하도록 만든 이름
  • 프로퍼티 : 스타일 속성 이름. CSS3에는 약 200개 정도의 프로퍼티가 있다.
  • 값 : 프로퍼티의 값
  • 주석문 : /*...*/주석문. 여러 줄에 걸쳐 아무 위치나 사용 가능
*셀렉터, 프로퍼티, 값은 대소문자를 구분하지 않는다.

CSS3 스타일 시트 만들기

    HTML문서에 CSS스타일시트 만드는 3가지 방법
  • <style>태그에 스타일시트 작성 <style>태그는 <head>태그 내에서만 작성 가능하며, 작성된 스타일시트는 웹 페이지 전체에 적용된다.
  • style 속성에 스타일시트 작성
  • 스타일시트를 외부 파일로 작성 후, 파일을 불러와 사용 두가지 방법 사용 가능
    <link href="filename.css" type="text/css" rel=stylesheet">
    @import url("filename.css")

CSS3 규칙

-스타일 상속

부모태그
자신을 둘러싸는 태그. 부모 태그로부터 스타일시트를 상속받는다

<p>태그의 스타일시트 color:green
<em>은 <p>의 자식이다.


-스타일 합치기와 오버라이딩

    각 태그에는 4가지 종류의 스타일시트가 동시에 적용될 수 있다
  1. 브라우저의 디폴트 스타일시트
  2. .css 스타일시트 파일에 작성된 스타일시트
  3. <style>태그에 작성된 스타일시트
  4. style속성에 작성된 스타일시트
*아래로 내려갈수록 우선순위가 높다.

<style>에서 작성한 ivory색 배경보다 <p>의 style속성에 작성한 orange색 배경이 우선 적용된다.

셀렉터

HTML태그의 모양을 꾸밀 스타일시트를 선택하는 기능
태그 이름 셀렉터 태그 이름이 셀렉터로 사용되는 유형
셀렉터와 같은 이름의 모든 태그에 CSS3 스타일시트 적용

*예시 : h3, li {color:brown;}

class 셀렉터 점(.)으로 시작하는 이름의 셀렉터
HTML태그의 class속성으로만 지정 가능
여러 태그를 단체로 동일한 CSS스타일을 적용할 때 적합

.red {color:red;}셀렉터를 적용

ID 셀렉터 #으로 시작하는 이름의 셀렉터
HTML태그의 ID속성으로만 지정 가능
여러 태그 중 특정 태그에만 CSS스타일을 적용할 때 적합

#red {color:red;}셀렉터를 적용

셀렉터 조합하기 2개 이상의 셀렉터를 조합하여 사용할 수 있다.
자식 셀렉터
부모 자식 관계인 두 셀렉터를 '>'기호로 조합
dd > small {color:dodgerblue;}
자손 셀렉터
자손 관계인 2개 이상의 태그 나열
dt small {color:dodgerblue;}

전체 셀렉터와 속성 셀렉터
-속성 셀렉터
HTML 태그의 특정 속성에 대해 값이 일치하는 태그에만 스타일을 적용
input[type=text]{color:red;} -> type속성값이 "text"인 <input>태그에 적용

-전체 셀렉터
와일드카드(*)를 사용하여 웹 페이지의 모든 태그에 스타일을 적용
ex) *{color:red;}

가상 클래스 셀렉터 어떤 조건이나 상황에서 스타일을 적용하도록 만드 셀렉터
->40개 이상의 가상 클래스 셀렉터들이 있다.
li:hover {color:gray;}
    자주 쓰이는 가상 셀렉터
  1. visited : 방문한 링크에 스타일 적용
  2. hover : 마우스가 올라갈 때 스타일 적용
  3. first-letter : 블록형 태그의 첫 글자에 스타일 적용

색 표현

CSS3에서는 3가지 방법으로 색을 표현할 수 있다.

  • 16진수 코드로 표현 -> #8A2BE2
  • 10진수 코드와 RGB()로 표현 -> rgb(138, 43, 226)
  • CSS3 표준에 지정된 색 이름으로 표현 -> blueviolet
CSS3의 색 관련 프로퍼티 : color, background-color, border-color

텍스트

CSS3로 문단 들여쓰기나 정렬 등 출력 모양을 제어할 수 있다.

{text-indent:1em;} -> 1글자 들여쓰기

{text-align:center;} -> 중앙 정렬

{text-decoration:underline;}

{text-decoration:line-through;}

CSS3의 표준 단위

폰트

폰트에는 Serif, Sans-Serif, Monospace 3가지의 유형이 있다.

-폰트의 프로퍼티

  • font-family : 구체적인 폰트 이름 지정
  • font-size : 폰트 크기

    font-size:40px; -> 40픽셀 크기
    font-size:medium; -> 중간 크기
    font-size:1.6em; -> 현재 폰트의 1.6배 크기

  • font-style : normal, italic, oblique 중 하나를 지정

  • font-weight : 100~900사이의 수, normal, bold, lighter에서 지정

-단축 프로퍼티, font

font의 style, weight, size, family를 순서대로 지정
